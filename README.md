# ZIVY Expense Tracker

A professional desktop application built with Python and PyQt6 for tracking personal finances. 

## Features
- **Dashboard Analytics:** Interactive bar, line, and pie charts (using PyQtGraph).
- **Transaction Management:** Add, edit, and delete income and expenses easily.
- **Dark Theme Interface:** A modern, sleek UI matching the ZIVY product line.
- **Local Database:** All data is securely stored locally using an SQLite relational database.

## Architecture
- **Language:** Python 3.x
- **Framework:** PyQt6
- **Charting:** PyQtGraph + NumPy
- **Database:** SQLite3

## How to Run
1. `python -m venv venv`
2. `.\venv\Scripts\Activate.ps1`
3. `pip install -r requirements.txt`
4. `python src/main.py`
